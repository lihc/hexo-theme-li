# hexo-theme-li

示例网站：[Demo](http://hi.liruiyi.xyz)

使用 master 分支构建

```shell
git clone -b master https://github.com/lihc/hexo-theme-li.git themes/li
```

使用 dev 分支构建

```shell
git clone -b dev https://github.com/lihc/hexo-theme-li.git themes/li
```
